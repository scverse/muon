# Contributing

Contributions to `muon` are welcome, whether they are issues, pull requests or discussions.

Please see the [contributing guide](docs/contributing.md) for how to set up a development environment, run the tests, build the documentation and open a pull request.

The main entry point for a contribution is an [issue](https://github.com/scverse/muon/issues).
Please use issues to discuss the change you wish to make or the functionality you want to add to `muon`.
For a more in-depth discussion you can also use [discussions](https://github.com/scverse/muon/discussions).
